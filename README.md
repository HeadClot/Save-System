# Save-System
A Save system for Unity aimed at artists and designers.
See the SaveSystem Folder for more indepth documemation on how to use this. 

**Why did I make this?**

I did not want to pay for Easy Save 3 on the asset store and I needed a save system that is artist focused and easy to use.

I hope you enjoy using this save system
and If you run into any problems file an issue.
