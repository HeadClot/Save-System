# What is uEasy-Save-System
A Save system for Unity aimed at artists and designers.
See the SaveSystem Folder for more indepth documemation on how to use this. 

**Why did I make this?**

I did not want to pay for Easy Save 3 on the asset store and I needed a save system that is artist focused and easy to use.
The scope for this is limited. It is intended for Single player games. I have not tested this with local only couch coop games and It does not support playerprefs.
I do not see any reason why it wont work though with a local only couch coop game. 

I hope you enjoy using this save system. I made it as easy to easy to use as I could. 
If you run into any problems or hurdles file an issue.
